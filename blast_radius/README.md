# Blast Radius MCP Server

Current implementation status: Milestones M1–M3 complete (schemas, validation, deterministic IDs/cache, Tool1 AST engine).

## Setup

```bash
pip install -e ".[dev]"
```

## Usage

```bash
blast-radius-mcp
```
